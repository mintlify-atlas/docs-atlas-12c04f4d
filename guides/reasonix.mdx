---
title: "Run DeepSeek with Reasonix, the DeepSeek-native agent"
description: "Run Reasonix, the DeepSeek-native terminal agent with cache-first architecture and flash-first cost control, using npx — no install required."
---

Reasonix is a DeepSeek-native coding agent built specifically for the DeepSeek API. It uses a cache-first loop, flash-first cost control, and automatic tool-call repair — and talks directly to `api.deepseek.com` without a translation layer. No global install is required; you can run it with `npx` from any project directory.

<Steps>
  <Step title="Install Node.js">
    Install [Node.js](https://nodejs.org/en/download/) 20.10 or later. Windows users also need [Git for Windows](https://git-scm.com/download/win).
  </Step>

  <Step title="Get a DeepSeek API key">
    Go to the [DeepSeek Platform](https://platform.deepseek.com/api_keys) and create an API key.

    <Note>
      You do not need to set an environment variable in advance. The first time you run Reasonix, a built-in wizard will prompt you for the key and persist it to `~/.reasonix/config.json`.
    </Note>
  </Step>

  <Step title="Run Reasonix in your project">
    Navigate to your project directory and launch Reasonix with `npx`:

    ```bash
    cd /path/to/my-project
    npx reasonix code
    ```

    Reasonix defaults to **DeepSeek-V4-Flash** for cost-efficient iteration. Use the following slash commands inside the TUI to control the model:

    | Command | Effect |
    |---------|--------|
    | `/pro` | Switch to **DeepSeek-V4-Pro** for the next turn only |
    | `/preset max` | Use **DeepSeek-V4-Pro** for the entire session |
    | `/help` | Show the full slash-command reference |
  </Step>
</Steps>
