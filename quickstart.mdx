This quickstart walks you through getting a DeepSeek API key and choosing the first tool to connect it to. Every guide in this collection follows the same three-step pattern: install, configure with your API key, and run. You only need a few minutes to go from zero to a working DeepSeek-powered agent.

<Note>
  The same DeepSeek API key works with every tool in this collection. Get it once from the [DeepSeek Platform](https://platform.deepseek.com/api_keys) and reuse it across Claude Code, OpenCode, GitHub Copilot, AstrBot, and all the other guides here.
</Note>

<Steps>
  <Step title="Get an API key from DeepSeek Platform">
    Sign in to the [DeepSeek Platform](https://platform.deepseek.com/api_keys) and create a new API key. Copy the key and store it somewhere safe — you'll need it in the next step.

    The platform gives you access to both models used throughout these guides:

    | Model | Best for |
    |-------|----------|
    | **DeepSeek-V4-Pro** | Complex reasoning, deep code analysis, multi-step tasks |
    | **DeepSeek-V4-Flash** | Fast iteration, cost-efficient workflows, sub-agent tasks |
  </Step>
  <Step title="Choose your tool">
    Pick the category that best fits how you work:

    <CardGroup cols={2}>
      <Card title="Terminal coding agents" icon="terminal" href="/guides/claude-code">
        Run DeepSeek inside Claude Code, OpenCode, Crush, Reasonix, and more — all from the command line.
      </Card>
      <Card title="Editor & IDE integrations" icon="code" href="/guides/github-copilot">
        Power GitHub Copilot, Kilo Code, and WorkBuddy with DeepSeek models directly inside VS Code.
      </Card>
      <Card title="Chat platform agents" icon="comments" href="/guides/astrbot">
        Deploy DeepSeek agents on Feishu, Telegram, WeChat, and other chat platforms.
      </Card>
    </CardGroup>
  </Step>
  <Step title="Install and configure">
    Open the guide for your chosen tool and follow the three steps it contains:

    1. **Install** the tool using the provided command or download link.
    2. **Configure** your environment with the DeepSeek API key and base URL.
    3. **Run** the tool and start your first session.

    Each guide includes exact commands for your operating system (Linux, macOS, and Windows where applicable).
  </Step>
</Steps>
